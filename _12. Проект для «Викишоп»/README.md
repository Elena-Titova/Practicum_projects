# Описание проекта "Проект для «Викишоп»" 
***проект модуля "Машинное обучение для текстов"***

## Цель исследования

Интернет-магазин «Викишоп» запускает новый сервис. Теперь пользователи могут редактировать и дополнять описания товаров, как в вики-сообществах. То есть клиенты предлагают свои правки и комментируют изменения других. Магазину нужен инструмент, который будет искать токсичные комментарии и отправлять их на модерацию.

Необходимо обучить модель классифицировать комментарии на позитивные и негативные. В нашем распоряжении набор данных с разметкой о токсичности правок.

Построим модель со значением метрики качества F1 не меньше 0.75.


## Описание данных

Данные находятся в файле /datasets/toxic_comments.csv. 
Столбец text в нём содержит текст комментария, а toxic — целевой признак.


## Ход исследования

В рамках проекта была выполнена предобработка данных, обучены модели с целью классифицировать комментарии на позитивные и негативные.


## Итоги исследования

По ТЗ необходимо было достичь показателя F1 не менее 0.75. 
Нам удалось достичь показателя F1-меры на тестовой выборке для Logistic Regression: 0.76
